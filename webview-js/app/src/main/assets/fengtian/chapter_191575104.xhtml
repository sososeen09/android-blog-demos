仙帝之上，是罗天帝仙，罗天帝仙上……则是罗天道仙！

罗天帝仙难，罗天道仙……难上加难！！

那才是，仙这个境界的最终极致，是可以撼动古境巅峰的存在，是能让道境强者，也都动容的存在！

跨越一个大境，如同超级强者，这样的人，无论在任何地方，任何时代，都将被众生瞩目！

可惜，罗天帝仙，极为稀少。比道境还少，罗天道仙，更为稀少。纵观仙界历史，古往今来。只有九人！这九人的地位，哪怕是三大至尊也要礼遇。

因为，仙界自古有传说，罗天道仙，是仙界之根，一旦能成为至尊……则是星空之巅！

而九封至尊，他是以罗天帝仙成就至尊，故而才会成为至尊之首。

若是以罗天道仙成就至尊。此事……古往今来，没有人做到，太难太难，成为罗天道仙难，罗天道仙在古境内，灯灭人灭的几率，超出其他古境无数倍，到了最后，几乎十死无生，连一线生机都不可见。如此……又怎能踏古摘道！

古往今来，这九个罗天道仙，有五人死亡在了古境。灯灭人灭，其他四人……则是在那场浩劫中，要么失踪，要么陨落。

曾经也有至尊去追寻仙界中，罗天仙是如何出现，可却无迹可寻，唯有一个事情，可以得到确定……若是自身血脉内，曾经有人成为罗天仙。哪怕只是罗天帝仙，那么他的后人。成为罗天帝仙的可能性，就会极大！

而若是血脉中曾经有先祖是……罗天道仙。那么其后人，成就罗天仙的可能性，会更大！

这几乎是脉脉相传，甚至若追寻最古老的历史，似乎……来源于……九个姓！

至于更深远，曾经那位至尊，在追寻时，却突然心悸，立刻放弃，似乎有某种意志，阻止追寻真相！

如今距离仙界崩溃，已过去了很久很久，在孟浩之前，连仙帝都没有出现过，罗天仙，也自然成为了传说，甚至已被太多人以往。

可在这一刻……

在这枚涅槃果与孟浩眉心碰触的瞬间，孟浩身体猛地震动，他的脑海中，掀起了惊天的风暴，轰隆隆的巨响，仿佛要将孟浩的识海直接炸开。

他的肉身，在这一瞬，竟膨胀起来，眨眼的功夫，就达到了三丈之高，还在继续，五丈，八丈……

随着身体的膨胀，那种血肉撕裂的感觉，让孟浩更为颤抖，若非他坚定的意志，此刻必定会嘶吼，他的全身流下的已不仅仅是剧痛的汗水，还有鲜血从汗毛孔内溢出。

血与汗融合在一起，使得孟浩全身，如成血人！

可偏偏，在这剧痛中，一股极致的肉身之力，轰然爆发，这力量之强，好似在孟浩身上突破了某种存在的枷锁，使得孟浩整个人，仿佛一下子超出了某种限制，他的心脏砰砰加速跳动，他的身体在不断地颤抖，但却有强悍的让他心惊的力量，正在疯狂的从身体内滋生出来。

这还仅仅是肉身，随后，一股风暴在孟浩体内爆发，他身体踉跄后退，喷出鲜血，这风暴太强，太过恐怖，在这爆发中，掀起了孟浩体内的仙力，让他忽然意识到，身体的膨胀，只是一个开始而已，真正的蜕变，是在这修为上！

轰轰轰轰！

孟浩身体不断退后，鲜血一口口喷出，他的身体成为了血人，他的体内风暴更为滔天，似要将孟浩的身躯撕裂，似要将他体内的一切经脉，全部碎灭。

这力量太强，更让孟浩觉得心惊的，似乎这力量不是来自涅槃果，而是来自……他的血脉之中，而涅槃果，仿佛是一个钥匙，打开血脉内，这股恐怖力量的钥匙！

“这……”孟浩呼吸急促，退后时，他的身体滔天轰鸣，外人听不到，可孟浩自身却震耳欲聋，紧接着，他清晰的察觉到，自己体内的一百二十三条仙脉，竟在这一瞬，齐齐碎开，居然……融合在了一起，成为了一条仙脉！

唯一的一条仙脉，在他的身体内，构架出了一个完美的循环！

在这个循环凝聚出来的刹那，孟浩的修为，前所未有的疯狂爆发，眨眼的功夫，就超出了之前仙帝太多太多，他的气势崛起，他的气息恐怖，他的爆发滔天。

轰鸣回荡，孟浩终于无法承受，仰天发出一声惊人的嘶吼，在这嘶吼中，似体内的风暴宣泄出来，向着四周轰隆隆的横扫，那些阁楼残骸，直接飞灰，那些洞府石室，瞬间消散，鹦鹉睁大了眼，露出震撼与无法置信。

“罗天仙！！居然是罗天仙，该死的，这孟浩居然有这种血脉，他的先祖是谁，方家，方家，该死的，我怎么想不起来！！”鹦鹉尖叫中急速后退，翅膀猛的一挥，将那些早已被震撼的目瞪口地的妖修笼罩。

皮冻也在一旁颤抖，它骇然的发现，这一刻的自己，居然有了像孟浩膜拜的冲动，那是刻在轮回中不灭的痕迹！

苏嫣愣在那里，脑海嗡鸣，她呆呆的看着孟浩，神色内露出骇然。

就在这时，孟浩仰天再次一声大吼，在这吼声中，他的身体上瞬间浮现了青色！

这青色，是至高无上的颜色，是仙界内，尊贵的颜色！

因为青色……代表了罗天！

而罗天……代表了无限！

更是在这个时候，孟浩的身上，青光辉煌，隐隐间，一股比仙帝更强的帝意，带着无上的霸道，轰隆隆的在孟浩身上崛起。

仙帝的帝意，如果比喻成一片大地上一个国度的帝皇，那么这一刻在孟浩身上爆发的气势，那是……以一颗星辰为界的国度的霸道！

眼看这帝意更为滔天，可就在这一瞬，孟浩双眼直接黯淡，他身体如瘪了一下，那正在爆发的气势，那似乎再迈出一步，就可踏入到罗天帝仙的境界，在这一刹那，瞬间消散。

临昏迷前，他唯有苦笑，自己的这一次尝试，终究是鲁莽了一些，可他能感受到，融合第二枚涅槃果的境界，那种强大……让他心动，让他痴狂，让他渴望。
