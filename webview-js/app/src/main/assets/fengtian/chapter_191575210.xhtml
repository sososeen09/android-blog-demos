第1165章 全族来拜！

天云上人，形神俱灭！

随着他的死亡，那四个穿着黑色铠甲的身影，向着孟浩一拜之后，身体化作长虹，直奔天空漩涡而去，这漩涡早已停止了转动，看起来如同一个静止的黑洞，当这四道身影冲入其内后，这黑洞漩涡刹那就消失。

随着散去，星空的波纹不再出现，虚无不再震动，来自于入道的威压，也在这一刻彻底的散去，一切都恢复如常。

唯独……这天云集，从此改名，成为方家，成为了方家的一处据点所在。

方守道深吸口气，神色内的激动慢慢被藏起，他深深的看了孟浩一眼，右手抬起一挥，立刻这陨石群四周的光幕，顿时消失，不再封印那些传送阵，使得此地的修士，都一个个心惊肉跳，赶紧离去。

临走前，他们看向孟浩的目光，带着恐惧，更有强烈的忌惮，甚至在他们看去，这一刻的孟浩，根本就不是仙境，而是足以与老一辈的强者，相提并论的大能之辈。

随着他们的离去，可以想象，要不了多久，有关天云上人得罪孟浩，而被斩杀的事情，就会传遍整个第九山海。

而也注定了，孟浩的名字，会再次横扫第九山海，让所有家族宗门，全部震撼，他们会深刻的明白，所有人，已不能再将孟浩看成是小辈，而是要将他看成是足以与准道等同，甚至还需要更高的位置。

方家的姿态。已说明了一切，任何人敢对孟浩有丝毫恶意，整个方家。都将爆发。

那些原本是天云集的穿紫海长袍的弟子，此刻一个个颤抖，他们早就选择了归降，会被方家整合，有太多的方法与手段，可以让他们从此之后，对方家效忠。

此刻。随着众人的相继离去，留在这陨石群中的。就只剩下了方家的族人，一共来了数百族人，此刻一个个都看向孟浩时，带着狂热。

那是……他们的少族！！

可以与准道一战。甚至击杀了准道的少族，这样的修为，这样的身份，足以让所有方家族人，都心神震动，为之狂热。

更重要的是，因每一个血脉族人体内道种的存在，使得方家的族人，在天然上。就对孟浩这里有尊敬，这是连他们自己都很难察觉的潜意识。

这潜意识原本不强烈，可此刻在孟浩斩杀准道的气势下。顿时爆发开来，那一道道狂热的目光里，蕴含了对孟浩的认可。

至于那几个长老，此刻内心也在震撼，相互看了看后，忽然都将目光落在了四周的集市上。这一处曾经的天云集市，已具备了规模。本就是一块肥肉，无论家族内谁来镇守这里，都会有太大的好处。

“浩儿，这处集市，你觉得家族里，谁来管理为好？”就在这时，大长老方通天，忽然开口，他话语一出，立刻点出了这集市的关键所在，更是告诉了那些四周的长老，孟浩……才是此地最有话语权的一位。

孟浩看着天云上人消失的地方，半晌之后他转身，向着方守道以及大长老方通天一抱拳。

“祖爷爷，大长老，我觉得这处集市，让方西来管理为好，不然他在家族中历练不多，有了这处集市，也可让他成长。”孟浩笑着开口，这处集市，他之前在决定将其改名方家集市时，就想到了方西，琢磨着这处集市价值不小，与其给了别人，不如给方西。

“方西那孩子不错，就给他吧”方守道和蔼一笑，向着孟浩柔声开口，对他而言，区区一处集市，孟浩既然开口，他就没考虑过拒绝，更不用说孟浩的身份，就算是他，也都内心震动。

只不过是看到孟浩似乎并没有要将身份说出的意思，且此事实在太大，一旦传出，必定会撼动整个第九山海，甚至对于整个山海界，都将是一场格局的变化与轰动。

同样的，对于罗天家族之事，方守尽管了解一些，可却不是很详细，他需要与孟浩沟通，才可配合，使得家族用最快的速度崛起，成为真正的罗天家族，想到这里，方守道看向孟浩时，越发的和蔼。

“浩儿，走，咱们先回东胜星，你有些时间没回去了，让祖爷爷检查一些你的修为战力。”方守道哈哈一笑。

“祖爷爷，回东胜星不急，晚辈要先去南天星，拜见父母。”孟浩抱拳开口，抬头时，看向方守道。

方守道一愣，仔细的看了看孟浩，若有所思，只是微微思索，就立刻笑着点头。

“不愧是我方家的麒麟子，不但盖世无双，更是孝子，你说的没错，是老夫疏忽了，本就是要先去南天星才对。”

“你父亲那里，培养出了我方家如此麒麟子，更是为此被派出镇压南天星，他的功劳之大，为家族贡献之大，付出的辛苦之多，方家……难以报答。”

“老夫决定，立方秀峰，为家族族长！！哪怕秀峰不在东胜星，可家族之修，也要以其为尊！

老夫将亲自护法，这一次，去帮助秀峰，突破修为，从古入道！”方守道大袖一甩，立刻传出家族的封命。

“通天，传命家族，但凡仙五境以上族人，即刻从家族内前往南天星，所有长老必须来临，去南天星，举办我方家族长大礼！”

“老夫要让整个第九山海所有宗门势力都知晓，方秀峰就是我方家的族长，要让整个第九山海，全部来拜！”

随着方守道的声音传出，四周所有方家族人，全部脑海轰鸣，那几个长老更是倒吸口气，立方秀峰为族长，此事太大，要知道就算是方守道，也只不过是代族长而已。

族长，一族之长，无论是地位还是身份，都将滔天而起，而作为方家的族长，其重要的程度，对于第九山海来说，那就是巨头一般的人物。

这是方守道对孟浩释放的善意，更是因曾经的事情，给予的补偿，还有则是对孟浩这里的重视与在意。

如此一来，孟浩就当之无愧，名正言顺的成为少族！

而这，只是第一份大礼，第二份，就是他会出手，甚至方言墟也都相助，两大道境齐力，去帮助方秀峰踏入道境！

有他们帮助，再不惜家族付出庞大的代价，是可以做到，将入道的危险，降低到最少，若这样的话还不能成功入道，那么已不是方家的原因，而是方秀峰自己了。

而这种倾全族之力去帮助，整个第九山海，能受到如此待遇者，屈指可数！

这两份大礼之重，就算是孟浩，此刻听到后也是深吸口气，看向方守道时，他抱拳，深深一拜。

至于几个嫡系的长老，此刻振奋，齐齐向着方守道一拜，转身时，立刻随着方通天，飞向传送阵。

临踏入阵法前，方通天深深的看了孟浩一眼，他之前做错了，如今的这些事情，是他对孟浩的弥补。

“浩儿，此事不急，来来来，我们先走一程，等他们都到了后，整个方家聚集在南天星外，与你一起，拜见我方家族长！”方守道哈哈一笑，看向孟浩时，孟浩立刻走来，二人身体一晃，直奔传送阵，刹那间，消失无影。

自然是方守道有太多的事情，想要与孟浩单独密探，想要知晓答案。

他们尽管离去，可方家这里的族人，却全部沸腾，随着一个个传送回东胜星，随着这个消息传遍东胜星的方家，整个方家，为之轰动起来。

不是没有人质疑，可却不敢违反方守道的命令，而方言墟也走出闭关之地，以其身份，让整个方家在短短的时间内，就立刻整合，足有上百万修士，一批批盛装传送，直奔南天星。

方家的如此动作，也传出了东胜星，使得所有宗门家族，全部都知晓，都在震惊。

也就是七天的时间，南天星外，传送之光滔天，在那一道道光芒后，方家的族人，全部都神色严肃，身影一一出现，随着到来之人越来越多，直至最终，几乎看不到边际。

每个人都穿着家族的盛装，全部都站在星空，而在他们的前方，就是南天星。

方言墟站在众人之前，他身边是丹老，身后是所有长老，长老后是嫡系族人，四周所有族人，密密麻麻，气势如虹，让星空颤抖，让虚无震动。

也正是在这个时候，孟浩与方守道，在一道传送之光中现身，方守道看似如常，可实际上目中的狂喜与激动，无法掩盖与压制，这一点方言墟立刻就看到，他也看到了，方守道冲着自己点了点头。

尤其是，方言墟发现，方守道站着的位置，居然是在孟浩身后，差了半步的距离，看似没什么，可这一幕在方言墟眼中，代表了无上含义。

他的目中，也露出了激动。

孟浩站在所有方家族人的前方，他的身后是方家全族的近乎所有强者，他的内心激昂，他……要回家了！
