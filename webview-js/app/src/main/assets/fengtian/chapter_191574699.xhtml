第五卷 涅盘重生血八方 第671章 合作

几个呼吸后，阳魂圣双眼刹那露出果断，右手抬起一把拿起酒杯，放在嘴边，一口饮尽。

这杯酒喝完之后，他忽然身体猛的震动，脸上瞬间出现了青筋，神色更有一抹痛苦，但却在死死的忍耐，右手缓缓抬起，体内修为全力运转，他身边妇人面色一变，正要起身时，被阳魂圣目光阻止。

他呼吸急促，十多息后，右手猛的一指，立刻在他的指尖位置，有一抹蓝色的气息散出，这气息刚一出现，就显露出锐利之意，蓦然飞出，直奔一旁高塔墙壁，刹那消失时，一股滔天的剑气，直接在远处的天地间，惊天而起。

阳魂圣呼吸急促，做完这些，面色微微苍白，抬头时，盯着孟浩。

“你家乡人，都喝这种酒？”

孟浩神色如常，拿起青桐酒壶，喝下一大口，看了阳魂圣一眼。

阳魂圣面皮再次抽动，望着孟浩，看到孟浩没有丝毫异常，甚至又喝了一口时，长叹一声。

“林某修行至今，被人以一壶酒威胁，还是首次。”

“我请你喝酒，怎么会是威胁，我家乡人，的的确确都是喝这种酒，只不过我出来的匆忙，没有带出多少，等下次会去时，我再给你多拿点。”孟浩有些冤枉的说道。

阳魂圣沉默，他有些拿捏不准孟浩的来临，这壶酒内蕴含了剑气，这剑气之强，他之前就心惊胆颤，可见孟浩这里喝下去如常，似没有半点不适后，他不知觉得，对眼前的孟浩。在心里多了一份神秘。

“可吸引海妖狂暴，拥有这种剑酒，肉身更是惊人。能抵消我的冰寒之气……他还是岁月之修，此人的来临……摸不透。”阳魂圣内心暗道。抬头时，端起装着寒酒的杯子，喝下一口。

“你口袋里的妖心，兑换给我阳魂道吧，价格不会太高，但也不会少。”

孟浩沉吟片刻，看了阳魂圣一眼。

“三宗为何对这妖心，如此看重？”

“这对其他人来说。是秘密，可以你的修为，早晚也能自己察觉。”阳魂圣平静开口，又继续说道。

“三宗传承很久，具体多少年，我也记不清了，只知道宗门第一代老祖，参与了飓风壁障的布置，随后闭了死关，至今未出。

是生是死。无人知道，老祖留下了一件传承之宝，此宝需吞噬妖心。才可维持形神不灭，而三宗之修，所修功法，也与外人不一样，灵石对于我们来说，作用一般，但妖心则大不一样。

另外，最重要的是，三宗的护宗大阵。每一次运转，都需要惊人数量的妖心支撑。否则无法展开。”阳魂圣淡淡说道。

孟浩听完此话，笑了笑。

“怕是还有别的原因吧。”

“有。可不能告诉你。”阳魂圣瞪着孟浩。

“与飓风壁障内的，黄泉幽轮有关。”孟浩双眼一闪，微笑说道。

他话语一出，阳魂圣面无表情，看着孟浩。

“你不用试探，这是三宗隐秘，不可能告诉外人，不过你能知晓黄泉幽轮……倒是让本圣有些意外。”

话语如常，可阳魂圣的内心，在孟浩说出黄泉幽轮这四个字时，却是一震，对孟浩这里的神秘感，更多了一些。

就在这时，忽然的，阳魂圣储物袋内，有玉简闪烁，他一拍后取出，看了一眼，面无表情的放在了案几上。

“你不愿说就罢了，至于妖心……我可以兑换给你，我需要大量的灵石！”孟浩不再试探，右手直接将储物袋解下，放在了案几上。

“这里面有多少，你应该心里也都清楚，开个价吧。”孟浩目光扫过阳魂圣面前的玉简，笑着开口。

“价值五十亿下品灵石之物。”阳魂道淡淡说道，右手提起时，取出一枚戒指，向前一甩，这戒指化作一道长虹，被孟浩一把戒指。

他神识一扫，立刻看到这戒指内，密密麻麻，众多的灵石！

不是下品，也不是中品，全部都是上品灵石！

孟浩双眼蓦然收缩，如此多的上品灵石，按照孟浩所知，哪怕是大宗家族，想要拿出也并非易事，可看这阳魂圣的姿态，似……毫不在意。

孟浩内心嘀咕了几句，暗道原来眼前这个冷面的家伙，才是真正的有钱人，可他又很不解，区区一个阳魂道，怎么可能会有这么多灵石。

孟浩目光闪动，他想到了妖心。

“莫非，这妖心，蕴含了什么可以让这阳魂道，如此富有的原因？”

“交易完成，不送。”阳魂圣闭上了眼，不再理会孟浩，似对看一眼都觉得烦躁。

孟浩笑了笑，将这戒指收走后，起身正要离去，但走出两步时，他忽然脚步一顿，回头看向阳魂圣。

“既然你们阳魂道灵石这么多，或许我们可以合作一下。”

阳魂圣双眼蓦然睁开，看向孟浩。

“怎么合作。”

“你我联手，灭杀海妖，我帮你引海妖出现。”

阳魂圣目中刹那精光一闪，沉吟片刻后，缓缓开口。

“如何分配。”

“所获妖心，你九我一，至于灵石，你可给我九折兑换！”

“五折兑换！”阳魂圣立刻说道。

“不可能，引来海妖并非容易，此事我占主导，只是因你也出手，速度快一些，且这里毕竟是三环，属于你们三宗，否则的话，我大可自己去做。”孟浩摇头。

“你也知道这里是三环！”阳魂圣再次开口，二人针对如何分配，不断讨价还价，美妇在旁看着这一幕，掩口一笑，她有很多年，没有看到阳魂圣与人说这么多话了。

她了解自己的夫君，看似冷漠，可实际上一旦被他认可，便是挚友，若是不认可，哪怕再强之人，也休想让他说上三句以上的话语。

她可以看出，夫君与孟浩之间，看似不和，但实际上，二人都有些对彼此欣赏。

“我告诉你黑小子，你别敬酒不吃吃罚酒，我家乡的酒味道如何你品尝了，你说的五折兑换，不可能！”孟浩一拍桌子，震的酒杯与玉简都飞起。

阳魂圣冷哼一声。

“三环内，这几百年本圣说的算，没我同意，你能杀妖？”

二人怒视，你看着我，我看着你，美妇在旁笑了笑，起身为二人倒满了酒水，轻声开口。

“何必如此，五折也好，九折也罢，不如各退一步，七折可好？”

“好，那就七折！”孟浩想了想，点头认可，这里毕竟是三环，且孟浩觉得自己除非用杀招，否则的话，与这阳魂圣斗法，终究是差了一些，虽然彼此都奈何不了对方，但想到自己赚灵石时，总有人来打扰，也是闹心。

阳魂圣略有迟疑，也点了点头。

“黑小子，那就说定了，这几天咱俩就出去。”孟浩起身。

“本圣叫林涛！”阳魂圣咬牙开口。

“行了黑小子，我知道了。”孟浩一挥手，正要离去。

阳魂圣忽然双眼一闪，将案几上的玉简一挥，飞向孟浩，孟浩转身一把接住，看向阳魂圣。

“飞仙圣尊，海神圣尊，给我传了一个消息，你看一下吧。”阳魂圣拿起酒杯，喝下后闭目，不再说话。

孟浩神色一凝，神识融入玉简内，看清了其内信息后，他心中猛的一震，这信息里，包含了飞仙宗与海神宗的七万弟子，正在进入三环的路上。

还包含了飞仙圣尊与海神圣尊二人，也即将来临之事，另外在信息中，也有孟浩的画像，再就是邀请阳魂圣，帮助他们寻找孟浩。

最后，点出了王家第十祖即将来临，或许会找阳魂圣，许下好处，让阳魂道也参与进来。

孟浩面无表情，将这玉简扔出，漂浮在阳魂圣的身前。

阳魂圣接过玉简，收起后淡淡开口。

“韩风。”

话语一出，瞬间在他身前，韩风瞬移而来，恭敬一拜。

“海城内，抹去为师这位朋友在海城内的一切信息，不管他接触了多少人，不管有多少人看到他，你都去处理的干干净净。

包括这几天，离开城池的，也都扫干净。”阳魂圣平静说道。

韩风一愣，立刻低头称是，看了孟浩一眼，转身离去。

孟浩看着这一幕，双眼微闪，没有说话。

“飞仙圣尊与海神圣尊，度过飓风壁障，以他们的速度，需两个月的时间，至于那位王家的第十祖，就算是问道修为，但也至少要两个月。”阳魂圣看向孟浩。

“本圣会答应他们的要求，也会答应王家第十祖的要求，这样的话，才可以在我的范围内，让你的行踪不会那么容易露出。”

“多谢！”孟浩沉默片刻，抱拳一拜。

“本圣不是帮你，是帮你我的生意。”阳魂圣淡淡说道，干咳一声，看了孟浩一眼，忽然开口。

“二折兑换！”

“最多六折！”孟浩咬牙，看着神色如常，但目中明显有得意之色的阳魂圣。
