第五卷 涅盘重生血八方 第670章 危机逼近

在孟浩于这海城内，散出气息，将那七个老者挥退，这七人各自喷出鲜血时，于天河海三环外，四环中，此刻有一片黑色残云，电闪雷鸣间，出现在了三环飞仙宗外。

残云消散，化作一个老者，他穿着一身黑色的长袍，容颜沧桑，好似有岁月之感隐隐环绕，站在天空上，他低头看向下方海面，目中露出深邃之芒。

若是孟浩在这里，定可一眼认出，这老者……正是王家第十祖！

“不可再耽搁了，这一次……一定要取走他的道基，若这具分身还是失败，那么老夫只要将其定位，不惜封仙台察觉，也要本尊挪移而来，达成所愿！”老者淡淡开口，目中刹那露出锐利之芒，蕴含了杀伐与果断。

“整个四环，没有他的痕迹，我来之前推衍，尽管找不到他的具体位置，但可算出他还在天河海！”

“四环没有，外海也没有，那么他一定是进了……三环！”

“三环……问道禁区……”老者皱起眉头，身体蓦然下沉，直奔下发飞仙宗。

在他进入飞仙宗约莫一炷香后，老者走出离去，几乎是他刚刚离开，整个飞仙宗，蓦然震动，来自飞仙宗圣尊的封命，瞬间发出，几乎大半个飞仙宗弟子，齐齐飞出，直奔三环外，飞仙海城而去。

飞仙宗弟子数万齐出，每个人手中都有一枚玉简，上面有孟浩的画像，甚至还有一缕王家第十祖的神识。

随着飞仙宗弟子一道道长虹踏入飞仙海城，立刻翻查入三环的记录，但却没有找到丝毫线索，可随后。这数万弟子，就立刻展开飞仙宗全力，在一艘艘舟船中。横渡飓风壁障，欲进入三环。完成宗门之命。

与此同时，三宗之一的海神宗，王家第十祖也亲自到访，不知许下了什么承诺，当他离去时，海神圣尊亲自下命，数万海神宗弟子，拿着有孟浩画像的玉简。化作无数剑光，直奔海神城，在其内，一一筛查了出入三环的记录后，这数万海神宗弟子，与飞仙宗一样，踏入三环内。

甚至飞仙圣尊以及海神圣尊，这两位斩灵强者，也在之后离开宗门，踏入飓风壁障内。一起进入三环。

三大宗，此刻已有两宗被王家第十祖驱动，当王家第十祖来到阳魂道宗门时。因阳魂圣尊驻扎在三环内的三宗海城，其他人无法决定，这才没有出动，但却为王家第十祖，筛查了阳魂海城的进出三环的记录。

“此子狡猾多端，但若反向思索，越是没有他的记录，则越有可能，他在三环！”王家第十祖沉吟片刻。眼中睿智之芒一闪，离开了阳魂道宗门。自行踏入飓风壁障内。

“我已在三环中，布置了天罗地网。孟浩……在这两宗七万修士的搜寻下，老夫不信……你还能在三环里，不翼而飞！”

“飞仙圣以及海神圣，也在老夫的重诺下，前往三环内，这二人修为一般，但拿着他们的传承之宝，还是有些犀利之处。

没有人比他们更了解三环，若你还能逃走，老夫倒也服了你孟浩。”

“不但他们在找你，阳魂道，老夫也有把握说服，到时候三宗十万修士，找你一人……若还不够，老夫已让人发布重赏，届时整个三环内所有修士，都是老夫的耳目。

只要有一个人找到你，老夫瞬间就可察觉。”王家第十祖迈步间，走在飓风壁障内，所过之处，一切黑雾齐齐散开，露出道路，似不敢阻挡在王家第十祖的面前。

“这一次，老夫必须要成功，时间太久，季族已有怀疑……”王家第十祖面色有些阴沉，他在之前，怎么也没想到，区区一个小辈，居然让自己如此棘手，失败两次。

在飞仙宗与海神宗七万修士，纷纷踏入飓风壁障内，直奔三环时，三环三宗海城内，阳魂道妖心阁中，孟浩右手虚空一抓，将所有的妖心都收了过来。

那七个老者寒蝉若惊，面色苍白，不敢继续阻挡，内心的苦涩，早已弥漫心神。

“前辈，这些妖心可否兑换给本宗，前辈放心，本宗一定给出前辈满意的价格！”孙云亮呼吸急促，赶紧开口，目中露出火热，孟浩身上的妖心，在他看来，若是能从自己手中兑换，那么对他而言，将是一件天大的功劳。

这样的功劳，甚至可以让圣尊亲自出手，对自己修为提点，使自己有那么一些可能，踏入斩灵境界！

即便不是踏入斩灵境界，也足以让自己在宗门内，万众瞩目。

“你阳魂道一个宗门，可以吃下这批妖心？”孟浩看了孙云亮一眼。

孙云亮一愣，算了算后，面色有些难看，他这才想起，哪怕是以阳魂道全宗之力，欲兑换这些妖心，也并非容易。

他迟疑时，阁楼外有长虹急速而来，刹那临近后，一个瞬移，直接出现在了二楼，正是阳魂圣的弟子，他刚一出现，立刻就看到了那七个面色苍白的老者，随后目光一闪，落在了孟浩身上。

只是一眼看去，他就立刻判断出，眼前之人，就是师尊所说的前辈。

这纯粹是一种直觉，这中年男子深吸口气，立刻抱拳一拜。

“晚辈韩风，拜见前辈，奉家师之命，请前辈一叙。”

他在来到时，孙云亮立刻神色一变，赶紧上前拜见，那七个老者也在看到这中年男子后，顿时露出敬畏之意，齐齐拜下。

孟浩目光在这中年男子身上扫过，双目微微眯起，抬头时，目光看向外界，落在了海城中那座高塔上。

“你师尊是阳魂圣？”孟浩淡淡开口。

“家师正是阳魂圣。”中年男子低头恭敬说道。

“我在阳魂圣那里等你，你清点了灵石后，可以来见我，你有多少灵石，我就换你多少价值的妖心。”孟浩看了孙云亮一眼，转身向前走去，唯离紧张的跟在身后，那中年男子，听闻此话一愣，看向孙云亮。

孙云亮赶紧传音，没有说具体的事情，只说了一下孟浩身上的妖心数量，中年男子双眼猛地睁大，倒吸口气。

他忽然明白了，为何师尊让自己来这里。

“他怎么会有这么多妖心！”中年男子赶紧跟在孟浩身后，一路更为恭敬，带着孟浩，飞向高塔。

唯离一路紧张，越是靠近高塔，她就越是呼吸急促，从跟随孟浩至今，这短短的数日，对她来说，见识了这一辈子都无法看到的一幕幕。

尤其是方才的妖心，更是让唯离心脏都似要停止跳动，她之前知道孟浩有钱，可却没想到，居然是……这么有钱。

富可敌国或许不如，但能与大宗抗衡的财力，却毫不夸张。

不多时，四人来到了高塔外，孟浩回头看了唯离一眼。

中年男子立刻上前抱拳一拜。

“前辈放心，这位道友晚辈会安排招待。”

孟浩点了点头，见唯离也没有意见，转身时，踏空走向高塔，直接走上最高层，刚一临近，闪烁间，消失不见。

出现时，已在了高塔内。

黑袍男子阳魂道，盘膝坐在一张案几后，面无表情，其旁那位雍容华贵的妇人，正闻着一壶酒水，抬头看向孟浩时，温柔一笑。

“你请我喝酒，我也请你一杯。”阳魂圣忽然开口，话语间，那妇人已端着酒壶来到案几旁，拿出两个酒杯，分别倒满后，坐在了阳魂圣的身边，带着一抹好奇之意，看向孟浩。

孟浩一言不发，直接坐在了阳魂圣的对面，拿起酒杯，看了一眼后，一口喝下。

酒水瞬间入了口中，刹那就化作一片寒冰，好似要将孟浩全身直接冰封，隐隐间，好似就连生命之火，也要在这寒酒下熄灭。

甚至这寒气顺着经脉，刹那覆盖全身，将孟浩体内的修为，也都眨眼间直接冰封，还有神识，也在这一刻，变的缓慢下来，仿佛整个人欲陷入沉睡之中，一个念头都很难升起。

在孟浩的皮肤上，更是出现了寒霜，看其样子，似这一杯酒，就可以让他化作冰雕。

这杯酒，寻常斩灵喝下，必死无疑，但孟浩这里肉身实在是太过强悍，他修为神识哪怕冰封，但他的肉身，只是微微一震，刹那间，这震动就出现了千百次之多，带着某种符合大道的频率，只是十几个呼吸的时间，他的头顶就升起了一片白雾，升空时，瞬间整个高塔顶层的墙壁，传来咔咔之声，竟直接冰封。

这一幕，让那妇人双目露出凝重，阳魂圣面无表情，冷眼看向孟浩。

“酒不够烈，不如我的。”孟浩右手抬起在储物袋上一拍，这一次不再是拿出葫芦，而是直接把韩山的青铜酒壶取出，又拿出一个酒杯，倒满后向前一推，这酒杯直接滑到了阳魂圣面前。

“我请你。”孟浩淡淡说道。

阳魂圣面皮抽动了一下，目光落在那酒杯上，目中罕见的出现了一抹迟疑。(未完待续)
